  • AuthToken

    Encapsulates an authentication signature that can be used when users subscribe to private or presence channels.

    The authentication token will contain any userData that was provided during the signing request. In the case of encrypted channels, it will also contain a base-64 encoded sharedSecret.

    Attributes of an occupied channel that can be fetched from the Pusher HTTP API.

    The attributes that can be fetched for a channel is dependent on its ChannelType. Attempting to fetch an attribute for a channel that does not support it will result in an error.

    A Webhook that contains a list of WebhookEvent instances related to changes in the internal state of a Pusher Channels app, that were received as a POST request to a user-specified Webhook URL.
